    # 蒜蓉油麦菜

    • 材料
    • 配料
    • 制作步骤

    # 材料

    • 油麦菜（或其他青菜）
    • 大蒜

    # 配料

    耗油、盐、味精

    # 制作步骤

    1. 油麦菜切段，洗净；
    2. 将油麦菜焯水后捞出（水中可加适量食用油）；
    3. 开火倒油，将蒜拍碎后放入锅中炒香；
    4. 加入油麦菜，大火翻炒，加入耗油、盐、味精调味；
    5. 出锅装盘；
