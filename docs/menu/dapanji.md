# 大盘鸡

[[toc]]




### 材料
- 鸡腿
- 青椒
- 土豆
- 洋葱

### 配料
桂皮、草果、香叶、八角、干辣椒、豆瓣酱、啤酒、生粉、老抽、生抽、料酒、小葱、白胡椒粉


### 制作步骤
1. 鸡腿切块、加料酒、生抽、生粉、白胡椒粉、八角腌制2个小时，其他材料切块备用；
2. 开火倒油，油温七成热时加入豆瓣酱炒熟，然后加入鸡腿翻炒；
3. 加入料酒继续翻炒，待料酒完全蒸发后加入生抽和老抽继续翻炒；
4. 加入适量啤酒、然后加入足量的水；
5. 烧开后，加入草果、桂皮、香叶、干辣椒，中火炖10分钟；
6. 加入土豆，炖至土豆口感微微发软；
7. 大火收汁，加入适量的盐调整味道；
8. 加洋葱、青椒翻炒出锅装盘；
9. 撒上葱花芝麻点缀；
